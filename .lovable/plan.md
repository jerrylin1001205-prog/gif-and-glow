

# EmojiVerse — Premium Emoji Marketplace & Chat App

## 1. Authentication & User Profiles
- Sign up / Login with email (Lovable Cloud backend)
- User profile with avatar, display name, and language preference
- Support for 10 languages: English, Chinese, Spanish, French, German, Japanese, Korean, Arabic, Portuguese, Russian

## 2. Emoji Marketplace — Browse & Buy
- **Categories**: GIF emojis, Sound emojis, Animated/Moving emojis, Word-style emojis (text in different fonts/styles), Standard emojis
- Each category displayed in a visually separated section with tabs or filters
- Premium packs show prices; free packs are marked as free
- Purchase flow with a cart and checkout summary
- Smooth animations and a modern dark/light theme toggle

## 3. Emoji Creator — Make Your Own
- Users can create custom word-style emojis by typing text and choosing from different font styles, colors, sizes, and animations
- Preview before saving
- Save created emojis to personal collection

## 4. Starred / Favorites
- Star any emoji to save it to a personal "Favorites" section
- Quick access from profile or sidebar

## 5. Real-Time Chat
- 1-on-1 chat with other users
- Send text messages alongside any emoji (standard, GIF, sound, animated, word-style)
- Emoji picker integrated into chat input
- Real-time message updates via Supabase realtime

## 6. Weekly Leaderboard (排行榜)
- Top emoji creators ranked by popularity each week
- Users can opt in/out of showing their name on the leaderboard (privacy toggle in settings)
- Displays top 10 with avatars and stats

## 7. Custom Backgrounds
- Gallery of preset background themes users can pick from
- Option to upload their own background image
- Background applies to chat and/or the main app interface

## 8. Multi-Language Support
- Full app translated into 10 languages
- Language switcher accessible from settings and onboarding

## 9. Modern Premium UI
- Glassmorphism cards, smooth transitions, and gradient accents
- Responsive design for mobile and desktop
- Bottom navigation on mobile, sidebar on desktop
- Dark mode by default with light mode option
